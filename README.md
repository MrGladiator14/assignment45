# Hospital Neural Network Analysis

**Repository:** https://github.com/MrGladiator14/assignment45

## Project Overview

This project focuses on analyzing hospital records using neural networks to predict patient outcomes and identify patterns in medical data.

## Codebase Structure

```
assignment45/
├── .git/                     # Git version control
├── .venv/                    # Python virtual environment
├── hospital_records.csv      # Raw hospital dataset
├── hospital_records_cleaned.csv  # Cleaned and processed dataset
├── hospital_nn_analysis.py   # Main neural network analysis script
├── hospital_nn_improved.py   # Improved version with enhanced features
├── hospital_ds.ipynb         # Exploratory data analysis notebook
├── hospital_neural_network_analysis.ipynb  # Neural network analysis notebook
├── hospital_nn_results.png   # Results visualization from initial model
├── hospital_nn_improved_results.png  # Results from improved model
├── pyproject.toml           # Python project configuration
├── uv.lock                  # Dependency lock file
├── W8_Tuesday_DailyAssignment (1).docx  # Assignment document
└── README.md                # This file
```

## Key Components

### Data Files

- **hospital_records.csv**: Raw hospital patient records
- **hospital_records_cleaned.csv**: Preprocessed dataset ready for analysis

### Analysis Scripts

- **hospital_nn_analysis.py**: Initial neural network implementation
- **hospital_nn_improved.py**: Enhanced version with better architecture and performance

### Jupyter Notebooks

- **hospital_ds.ipynb**: Exploratory data analysis and visualization
- **hospital_neural_network_analysis.ipynb**: Interactive neural network analysis

### Results

- **hospital_nn_results.png**: Visualization of initial model results
- **hospital_nn_improved_results.png**: Results from the improved neural network
